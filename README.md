# 🚀 NextMoveAI

> AI-powered career guidance platform that helps students and job seekers prepare for placements through personalized roadmaps, interview preparation, aptitude practice, resume analysis, and AI assistance.

## 📌 Overview

NextMoveAI is an intelligent placement preparation platform built to simplify the journey from learning to getting hired. It combines AI-powered guidance with practical preparation tools to provide users with a personalized experience.

Whether you're preparing for internships, campus placements, or technical interviews, NextMoveAI helps you stay organized and focused.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/NextMoveAI.git

cd NextMoveAI
```

### Install dependencies

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd frontend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

GEMINI_API_KEY=your_google_gemini_api_key

EMAIL_USER=your_email

EMAIL_PASS=your_email_password
```
